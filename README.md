## 🕹️ 3D Prototype Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look around |
| Space | Jump |
| Shift | Sprint |
| F | Toggle fly mode |
| R | Toggle rain |
| M | Mute audio |
